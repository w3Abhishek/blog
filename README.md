# Minimal Personal Blog

A minimal, sepia-themed personal blog built with **Flask** and **Supabase**. Features a dark mode toggle, admin panel with WYSIWYG editor, and Vercel deployment support.

## Features

- **Minimal Design**: Clean, sepia-colored theme with a focus on readability.
- **Dark Mode**: Toggleable dark mode that persists user preference.
- **Admin Panel**: Secure admin area to create, edit, and delete posts.
- **WYSIWYG Editor**: Rich text editing for blog posts using Quill.js.
- **Supabase Backend**: Fast and scalable database for storage.
- **Vercel Ready**: Configured for easy deployment on Vercel.

## Tech Stack

- **Backend**: Flask (Python)
- **Database**: Supabase (PostgreSQL)
- **Frontend**: Jinja2 Templates, CSS3, Vanilla JS
- **Deployment**: Vercel

## Setup

### 1. Clone the repository
```bash
git clone https://github.com/w3Abhishek/blog.git
cd blog
```

### 2. Install Dependencies
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Environment Variables
Create a `.env` file in the root directory:
```bash
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
ADMIN_PASSWORD=your_admin_password
FLASK_SECRET_KEY=your_secret_key
```

### 4. Database Setup
Run the following SQL in your Supabase SQL Editor to create the `posts` table:
```sql
create table posts (
  id bigint generated by default as identity primary key,
  title text not null,
  slug text not null unique,
  content text,
  category text,
  tags text[],
  published boolean default false,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);
```

### 5. Run Locally
```bash
python app.py
```
Visit `http://127.0.0.1:5000` in your browser.

## Deployment

This project is configured for deployment on **Vercel**.

1.  Push your code to GitHub.
2.  Import the repository in Vercel.
3.  Add the Environment Variables in Vercel Project Settings.
4.  Deploy!
